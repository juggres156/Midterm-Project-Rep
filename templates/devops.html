DESIGN OF AN AUTOMATED WATERING SYSTEM
USING MACHINE LEARNING FOR MULTI-CROP FARMING

This project aims to create an automated multi-crop greenhouse watering system utilizing
machine learning that satisfies the client's demands while adhering to technical standards
and trade-offs based on recognized restrictions.

This gadget can irrigate crops automatically depending on their age and kind. Instead
of returning to the facility for water, the employee may configure the device to process.

Sensors monitor soil moisture to decide when to irrigate crops.
For each agricultural region (zone), a machine learning algorithm
was used to find trends in the watering history of the plants.
The soil moisture sensor will be used to monitor the soil.
